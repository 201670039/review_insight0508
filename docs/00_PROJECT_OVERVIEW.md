# AI Sentiment Analysis Service — Project Overview

## Goal
Build a BMW Corporate Design based AI sentiment analysis web service.

## Core Features
- Text input
- OpenAI sentiment analysis
- Positive / Negative / Neutral result
- Confidence percentage
- Analysis reason
- Modal result UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- AI: OpenAI API
- Database: Supabase
- Deployment: Vercel

## Architecture
Frontend → Backend API → OpenAI API → Supabase → Response UI
